Project Overview

펭돌이는 둥글고 포근한 실루엣과 작은 날개, 짧은 발을 가진 아기 펭귄 캐릭터입니다.

검정, 회색, 흰색을 기본 컬러로 사용하고 볼과 소품에 핑크 컬러를 더해 친근하고 사랑스러운 분위기를 표현했습니다.

캐릭터의 기본 형태부터 다양한 표정과 동작, 일상 대화에 활용할 수 있는 이모티콘까지 제작했습니다.

Cute Shy Cozy Playful

Character Turnaround

다양한 장면에서도 캐릭터의 형태가 일관되게 유지되도록 정면, 측면, 후면의 기본 구조를 제작했습니다.

Expression & Emoticon Set

사랑, 부탁, 피곤함, 거절, 인사와 화난 감정 등 일상 대화에서 활용할 수 있는 다양한 표현을 펭돌이 캐릭터로 제작했습니다.

Brand Application

K2 Safety Socks Package Design

펭돌이 캐릭터를 활용해 K2 Safety 등산 양말 패키지 콘셉트 디자인을 제작했습니다.

산악 배경과 겨울 등산을 준비한 펭귄 캐릭터를 조합하여 제품의 아웃도어 이미지를 표현했으며, 밝고 시원한 블루 톤을 사용해 활동적이고 청량한 분위기를 강조했습니다.

패키지 전개도부터 완성된 상자 형태, 실제 양말 제품을 활용한 광고 목업까지 제작하여 캐릭터가 브랜드와 제품에 적용되는 과정을 시각화했습니다.

Package Design Character Application Product Mockup Adobe Illustrator Adobe Photoshop Personal Concept Project

Personal concept project created for portfolio purposes.
포트폴리오 제작을 위한 개인 콘셉트 프로젝트입니다.

패키지 전개도와 완성된 상자 목업을 함께 구성하여 그래픽이 각 면에 자연스럽게 연결되는지 확인했습니다.

완성된 패키지와 등산 양말을 역동적으로 배치하여 실제 제품 광고에 활용되는 장면을 표현했습니다.

Design Style

둥근 실루엣과 굵고 부드러운 외곽선을 사용해 작은 화면에서도 캐릭터의 형태와 감정이 선명하게 전달되도록 디자인했습니다.

기본적인 무채색 캐릭터에 핑크 컬러를 포인트로 사용하여 따뜻하고 사랑스러운 인상을 더했습니다.
